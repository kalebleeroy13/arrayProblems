reverseArray()

function that takes an array as an argument & prints the array elements in reverse order.

reverseArray([1,2,3,4]);

//4

//3

//2

//1

maxReturn()

function that accepts an array of numbers then returns the largest number

maxReturn([1,2,3,]); //3

maxReturn([10,3,10,4]); //10

maxReturn([-5,100]); //100

isIdentical()

function that takes an array argument & returns true if all elements are identical

isIdentical([1,1,1,1,1]); //true

isIdentical([2,1,1,1,1]); //false

isIdentical(["a","c","a","f"]); //false

isIdentical(["d","d","d","d"]); //true

sumArray()

function that accepts array of numbers & returns sum of all numbers in array

sumArray([1,2,3]); //6

sumArray([10,2,10,4]); //27

sumArray([-10,35]); //25
